# MD Neyaz Ahmad Portfolio

Modern React + Vite portfolio website.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Upload this project to GitHub and import it on Vercel.

## Notes

- No `package-lock.json` included.
- No `node_modules` included.
- StudyNotion project is not included.
